 Stock Price Prediction using Machine Learning

This project predicts the stock price of **Coca-Cola (KO)** using Machine Learning techniques.
The model analyzes historical stock data and predicts future closing prices.



 Project Overview

Stock market prediction is an important financial application.  
In this project, historical stock data of Coca-Cola is analyzed and a **Random Forest Regressor** is used to predict future prices.



 Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  
- Yahoo Finance API  
- Google Colab  



 Features

✔ Fetches real-time stock data  
✔ Data preprocessing and cleaning  
✔ Moving Average calculation  
✔ Data visualization  
✔ Machine Learning prediction  
✔ Model evaluation using MAE  



 Machine Learning Model

- Algorithm: **Random Forest Regressor**
- Evaluation Metric: **Mean Absolute Error (MAE)**
- Input Features
- Open
- High
- Low
- Volume
- MA_20
- MA_50


 Output

- Stock price trend graph
- Actual vs Predicted price
- Future stock price prediction
